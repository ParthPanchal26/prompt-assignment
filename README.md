# Assignment 1 prompt
```text
You are an AI research analyst.

Provide a summary of the latest developments in Artificial Intelligence in 2025.

Return your response in STRICT JSON format only.
Do not include explanations, markdown formatting, or additional text.

The JSON structure must follow this schema exactly:

{
  "title": "string",
  "year": "2025",
  "key_developments": [
    {
      "area": "string",
      "description": "string",
      "impact": "string"
    }
  ],
  "major_companies_involved": ["string"],
  "future_outlook": "string"
}

Requirements:
- Ensure valid JSON.
- No trailing commas.
- No comments.
- No extra text before or after JSON.
```

# Assignment 2 prompt
```text
You are an academic research writer.

Write a structured research paper on the topic:
"The Effects of Social Media on Mental Health."

Follow these instructions strictly:

1. Divide the paper into the following sections:
   - Introduction
   - Literature Review
   - Findings
   - Conclusion

2. Each section must contain approximately 300 words.

3. After EACH section, add a subsection titled:
   "Section Summary"
   Provide exactly 3 concise bullet points summarizing that section.

4. Use formal academic tone.

5. Do not skip any section.

6. Do not combine sections.

7. Clearly label all headings.

Return the response in clean, well-formatted text with headings and bullet points.
```
